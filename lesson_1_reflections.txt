How can you use the commands git log and git diff to view the history of files?

O comando git log mostra todos os commits com ids, datas, autores e mensagem. Se
colocarmos a flag --stat ele mostra a quantidade de linhas adicionadas ou 
removidas.
O comando git diff compara duas commits. Temos que informar as ID encontradas 
com o comando git log.
